# 🎵 SONALI MUSIC BOT

A powerful, fast, and highly customizable Telegram Music & Utility Bot built with Pyrogram and PyTgCalls.

---

## 🚀 Commands Catalog / कमांड सूची

All commands supported by the bot categorized for easy access.

### 1. 🤖 Cloning & Multi-Bot / क्लोनिंग और मल्टी-बॉट कमांड्स
Create and manage your own cloned copies of this music bot with your own assistants!
* `/clone [Bot Token] [Assistant Session String]` - Clone this bot with your own assistant!
  * *Example / उदाहरण:* `/clone 8393838383:uege8sbz8sbsns BQBy3abc_def_ghi_jkl...`
* `/unclone` - Stop and delete your cloned bot.
* `/cloned` or `/clones` - Show status of your cloned bot (or all clones for Owner).
* `/broadcastclones [text]` - Broadcast a message through all cloned bots to all of their active dialogs (Owner only).

### 2. 🎵 Music Playback / संगीत प्लेबैक कमांड्स
* `/play` or `/vplay` - Play requested song/video in voice chat.
* `/cplay` or `/cvplay` - Play requested song/video in channel's voice chat.
* `/pause` or `/cpause` - Pause the currently playing stream.
* `/resume` or `/cresume` - Resume the paused stream.
* `/skip` or `/cskip` - Skip current playing track to next.
* `/next` or `/cnext` - Synonym for skip.
* `/end` or `/stop` or `/cend` or `/cstop` - Stop streaming and clear queue.
* `/seek` or `/cseek` - Seek the stream to a specific duration.
* `/loop` - Loop/Repeat the current playing track.
* `/speed` or `/cspeed` - Adjust playback speed.
* `/queue` or `/cqueue` - Check currently queued tracks.

### 3. 🛡️ Sudo / Owner Commands / सुडो और ओनर कमांड्स
* `/addsudo` - Add a user to sudo list.
* `/delsudo` or `/rmsudo` - Remove a user from sudo list.
* `/sudolist` or `/listsudo` - View list of sudoers.
* `/delallsudo` - Clear all sudoers (Owner only).
* `/gban` or `/globalban` - Globally ban a user from all groups.
* `/ungban` - Globally unban a user.
* `/gbannedusers` - List globally banned users.
* `/blchat` - Blacklist a group chat.
* `/unblchat` - Whitelist a blacklisted group chat.
* `/blchats` - List all blacklisted chats.
* `/restart` - Restart the bot.
* `/update` or `/gitpull` - Pull latest updates from Git.
* `/eval` - Execute Python code (Dev only).
* `/sh` - Run shell commands (Dev only).
* `/gadd` - Add the bot to multiple chats using assistant.

### 4. 👥 Group Management / ग्रुप मैनेजमेंट कमांड्स
* `/admins` or `/staff` - View list of group administrators.
* `/bots` - List all bots in the group.
* `/ban` - Ban a user from the group.
* `/unban` - Unban a user from the group.
* `/mute` - Mute a user.
* `/unmute` - Unmute a user.
* `/tmute` - Temporarily mute a user.
* `/unbanall` - Unban all banned users.
* `/unmuteall` - Unmute all muted users.
* `/unpinall` - Unpin all pinned messages.
* `/pin` - Pin a message.
* `/unpin` - Unpin last pinned message.
* `/pinned` - View currently pinned message.
* `/setphoto` - Set group profile picture.
* `/removephoto` - Remove group profile picture.
* `/settitle` - Change group title.
* `/setdiscription` - Change group description.
* `/user` - Show total members count.
* `/givelink` - Generate group invite link.
* `/mention` or `/all` - Mention/tag all group members.
* `/alloff` - Stop ongoing group tag.
* `/autoapprove` - Auto-approve group join requests.
* `/clearpending` - Clear pending join requests.
* `/vclogger` - Manage voice chat logging.
* `/lang` or `/setlang` - Change bot language.

### 5. 🛠️ General Utility / सामान्य उपयोगिता कमांड्स
* `/ping` - Check bot status and response latency.
* `/id` - Check User or Group Chat ID.
* `/stats` or `/gstats` - View system resources and database stats.
* `/song` - Download any song/audio directly.
* `/google` or `/gle` - Search google for queries.
* `/tgm` or `/tgt` - Upload files/media to Telegraph.
* `/whois` - Get profile details of any user.
* `/couples` - Find today's love couples in the group.
* `/speedtest` or `/spt` - Execute server network speedtest.
* `/carbon` - Create a beautiful carbonized image of text.
* `/stickerid` or `/stid` - Retrieve any sticker's file ID.
* `/packkang` - Copy stickers to your own pack.
* `/q` or `/r` - Create message quotations.
* `/tr` - Translate text.
* `/afk` - Go AFK (Away From Keyboard).
* `/gn` - Say Goodnight.
* `/shayari` - Get a random Hindi Shayari.
* `/shstop` - Disable automatic shayari in the group.
* `/ig` - Fetch Instagram profile or post details.
* `/reel` - Download Instagram reels.
