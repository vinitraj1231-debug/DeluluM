# 🎵 Bot Commands List / कमांड्स की सूची 🎵

इस Bot के अंदर उपलब्ध सभी कमांड्स की सूची नीचे दी गई है:

---

## 🎧 Music Commands (म्यूजिक कमांड्स)
ये कमांड्स म्यूजिक और वीडियो प्ले करने और उन्हें कंट्रोल करने के लिए हैं।

| Command | Description |
| :--- | :--- |
| `/play` या `/vplay` | वीडियोचैट पर मनपसंद ऑडियो/वीडियो स्ट्रीम करना शुरू करें। |
| `/playforce` या `/vplayforce` | चालू स्ट्रीम को रोककर तुरंत नया गाना प्ले करें। |
| `/cplay` | चैनल के वीडियोचैट पर ऑडियो स्ट्रीम शुरू करें। |
| `/cvplay` | चैनल के वीडियोचैट पर वीडियो स्ट्रीम शुरू करें। |
| `/cplayforce` या `/cvplayforce` | चैनल पर चल रही स्ट्रीम को रोककर नया गाना शुरू करें। |
| `/channelplay [username/id] / [disable]` | ग्रुप को किसी चैनल से कनेक्ट करें या डिस्कनेक्ट करें। |
| `/pause` | चालू स्ट्रीम को पॉज (Pause) करें। |
| `/resume` | पॉज की गई स्ट्रीम को फिर से शुरू (Resume) करें। |
| `/skip` | चालू गाने को छोड़कर अगला गाना प्ले करें। |
| `/end` या `/stop` | कतार (Queue) को साफ करें और स्ट्रीम को बंद करें। |
| `/player` | एक इंटरैक्टिव प्लेयर पैनल खोलें। |
| `/queue` | कतार में लगे गानों की सूची देखें। |
| `/shuffle` | कतार में लगे गानों को शफल (मिश्रित) करें। |
| `/seek [seconds]` | स्ट्रीम को आगे बढ़ाएं। |
| `/seekback [seconds]` | स्ट्रीम को पीछे ले जाएं। |
| `/song [song name/YT URL]` | यूट्यूब से कोई भी गाना MP3 या MP4 फॉर्मेट में डाउनलोड करें। |
| `/speed` या `/playback` | ग्रुप में प्लेबैक स्पीड एडजस्ट करें (केवल एडमिन्स के लिए)। |
| `/cspeed` या `/cplayback` | चैनल में प्लेबैक स्पीड एडजस्ट करें। |
| `/loop [enable/disable] / [1,2,3...]` | चालू गाने को लूप पर चलाएं। |

## 🛡️ Admin & Group Management Commands (ग्रुप मैनेजमेंट कमांड्स)
ये कमांड्स ग्रुप को मैनेज और सुरक्षित रखने के लिए हैं।

| Command | Description |
| :--- | :--- |
| `/ban [username/reply]` | यूजर को ग्रुप से बैन करें। |
| `/sban [username/reply]` | यूजर को ग्रुप से चुपचाप (Silent) बैन करें। |
| `/tban [username/reply] [time]` | यूजर को सीमित समय (m/h/d) के लिए बैन करें। |
| `/unban [username/reply]` | बैन किए गए यूजर को अनबैन करें। |
| `/kick` | यूजर को ग्रुप से किक (बाहर) करें। |
| `/kickme` | खुद को ग्रुप से बाहर करने के लिए। |
| `/mute [username/reply]` | यूजर को म्यूट (मौन) करें। |
| `/tmute [username/reply] [time]` | यूजर को सीमित समय के लिए म्यूट करें। |
| `/unmute [username/reply]` | यूजर को अनम्यूट करें। |
| `/pin` | किसी मैसेज को ग्रुप में पिन करें। |
| `/pinned` | ग्रुप के पिन किए गए मैसेज को देखें। |
| `/unpin` | पिन किए गए मैसेज को अनपिन करें। |
| `/staff` | ग्रुप के एडमिन्स और स्टाफ की सूची देखें। |
| `/bots` | ग्रुप में मौजूद सभी बॉट्स की सूची देखें। |
| `/settitle [title]` | ग्रुप का नाम बदलें। |
| `/setdescription [description]` | ग्रुप का विवरण (Description) बदलें। |
| `/setphoto` | ग्रुप की फोटो बदलें। |
| `/removephoto` | ग्रुप की फोटो हटाएँ। |
| `/unmuteall` | ग्रुप के सभी म्यूटेड मेंबर्स को अनम्यूट करें। |
| `/unbanall` | ग्रुप के सभी बैन मेंबर्स को अनबैन करें। |
| `/unpinall` | ग्रुप के सभी पिन मैसेजेस को एक साथ अनपिन करें। |
| `/zombies` | ग्रुप के सभी डिलीटेड अकाउंट्स (Inactives) को बाहर करें। |
| `/auth [username/id]` | किसी यूजर को बॉट के एडमिन राइट्स (Auth) दें। |
| `/unauth [username/id]` | यूजर को ऑथ लिस्ट से हटाएँ। |
| `/authusers` | ऑथराइज्ड (Authorized) यूज़र्स की सूची देखें। |

| `/status` | ग्रुप का स्टेटस देखें। |
| `/groupdata` | ग्रुप का डेटा प्राप्त करें। |
| `/groupinfo [username]` | ग्रुप की जानकारी देखें। |

## 👑 Sudo & Owner Commands (सुडो और ऑनर कमांड्स)
ये कमांड्स केवल बॉट के ऑनर या सुडो यूज़र्स के लिए हैं।

| Command | Description |
| :--- | :--- |
| `/broadcast` | बॉट के सभी चालू चैट्स में मैसेज ब्रॉडकास्ट करें। |
| `/blacklistchat [chat_id]` | किसी चैट को बॉट इस्तेमाल करने से बैन करें। |
| `/whitelistchat [chat_id]` | बैन चैट को अनबैन करें। |
| `/blacklistedchat` | ब्लैकलिस्टेड चैट्स की सूची देखें। |
| `/block [username/reply]` | यूजर को बॉट का इस्तेमाल करने से ब्लॉक करें। |
| `/unblock [username/reply]` | ब्लॉक किए गए यूजर को अनब्लॉक करें। |
| `/blockedusers` | ब्लॉक किए गए यूज़र्स की सूची देखें। |
| `/gban [username/reply]` | यूजर को ग्लोबली बैन करें। |
| `/ungban [username/reply]` | ग्लोबली बैन यूजर को अनबैन करें। |
| `/gbannedusers` | ग्लोबली बैन यूज़र्स की सूची देखें। |
| `/logs` | बॉट के लॉग्स (Logs) प्राप्त करें। |
| `/logger [enable/disable]` | बॉट की एक्टिविटी लॉगिंग ऑन/ऑफ करें। |
| `/maintenance [enable/disable]` | बॉट का मेंटेनेंस मोड ऑन या ऑफ करें। |
